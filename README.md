### Mi Contribución

Dentro de este proyecto, mis responsabilidades principales abarcaron tanto la gestión como el desarrollo técnico:

* **Planificación y Análisis:** Participación activa en la creación del plan de proyecto y en la redacción del Documento de Especificación de Requisitos de Software (SRS).
* **Desarrollo Backend:** Implementación de parte de la lógica de negocio y funcionalidades del sistema utilizando Java.
* **Desarrollo Frontend:** Creación e implementación de varias de las interfaces gráficas de usuario (GUI) con Java Swing.
* **Base de Datos:** Diseño y desarrollo de componentes de la base de datos del proyecto en SQL.
